# SmartInput 快速开始指南

## 📦 一句话介绍

**SmartInput** 是一个智能自适应的中英文输入法，自动识别输入内容是中文拼音还是英文，支持 **Ctrl+Shift** 快捷键切换，使用体验与搜狗输入法相同。

---

## 🚀 安装和运行

### 方式 1: 双击可执行文件（最简单）

直接双击 `dist/SmartInput.exe`，程序会在后台运行。

### 方式 2: 命令行运行

```cmd
d:\AI_Test\SmartInput\dist\SmartInput.exe
```

### 方式 3: 从源代码运行

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行程序
python main.py
```

---

## 🎮 基本操作

程序启动后，托盘会显示输入法图标。打开任何文本编辑器开始使用：

### 输入中文拼音

```
1. 输入: nihao
2. 显示: 你好, 尼号, ...（候选词）
3. 按 1: 输出"你好"
   或按 Space: 输出"你好"（根据候选词）
```

### 输入英文

```
1. 输入: hello
2. 程序识别为英文（无法匹配拼音规则）
3. 自动输出: hello
```

### 切换输入法

```
按 Ctrl+Shift 在中文和英文模式之间切换
托盘图标会显示当前模式:
  🇨🇳 中文模式
  🇬🇧 英文模式
```

| 快捷键 | 功能 |

## 🔧 常见问题

### 问题 1: 程序无法启动

**解决方案:**
- 确认 Windows 版本 7 或更高
- 确认是 64-bit 系统
- 尝试以管理员身份运行
- 检查磁盘空间（需 100MB+）

### 问题 2: Ctrl+Shift 无效

**解决方案:**
- 确认程序正在运行（任务管理器检查）
- 在记事本中测试
- 某些游戏可能拦截快捷键
- 重启程序

### 问题 3: 托盘图标不显示

**解决方案:**
- 检查 Windows 通知设置
- 图标可能被隐藏，点击"显示隐藏的图标"
- 重新启动程序

### 问题 4: 内存占用过高

**解决方案:**
- 这是正常的（50-100MB）
- 重启程序可能有帮助
- 检查是否有其他程序占用资源

### 问题 5: 无法输入中文

**解决方案:**
- 确保输入的是有效的拼音
- 检查是否在中文模式（托盘图标）
- 确认依赖库已正确安装
